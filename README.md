## 🔧 Кастомизация

### Изменение цветов:

Отредактируйте CSS переменные в `styles.css`:

```css
:root {
    --primary-color: #000;      /* Основной чёрный */
    --accent-color: #ff6b35;    /* Акцентный оранжевый */
    --text-color: #222;         /* Цвет текста */
    --light-text: #666;         /* Светлый текст */
    --bg-light: #f8f9fa;        /* Светлый фон */
}
```

### Изменение текста:

Просто отредактируйте нужный текст в `index.html`.

## 📊 SEO оптимизация

Для улучшения SEO добавьте:

1. meta description в `<head>`
2. Open Graph теги для соцсетей
3. Structured data (Schema.org)
4. Правильные заголовки h1, h2

Пример:

```html
<meta name="description" content="Max-бот автоматизирует продажи и принимает оплату в мессенджере MAX...">
<meta property="og:title" content="Max-бот — система продаж в мессенджере">
<meta property="og:image" content="image.jpg">
```
